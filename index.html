Tower-Defense by sb8244

Tower defense engine and basic game built on the Engine

This project, used as my design patterns final project, consists of an engine for tower defense games, as well as a simple game called "pixel tower defense" that utilizes the engine. As this was for my design patterns class, I focus on properly using and testing several different patterns including observer, adapter, factory, strategy, and singleton. I used a fibonacci heap to create an efficient implementation of the A* path finding algorithm.

Dependencies

Java Runtime Environment

Install

Execute the TDGame.jar in the TowerDefense folder by double clicking it or via terminal with "java -jar TDGame.jar"
